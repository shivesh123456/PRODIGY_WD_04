# Shivesh M - Personal Portfolio Website

## Overview
This is a modern, professional personal portfolio website showcasing web development skills, projects, and accomplishments. The website features a clean, responsive design with multiple pages and smooth navigation.

## Features

### Pages Included
1. **Home (index.html)** - Eye-catching landing page with:
   - Captivating headline and professional image
   - Skills preview section
   - Featured projects showcase
   - Call-to-action section

2. **About (about.html)** - Detailed background including:
   - Personal introduction and journey
   - Education timeline
   - Professional experience
   - Core values section

3. **Projects (projects.html)** - Complete project showcase featuring:
   - Project WD1: Foundation Web Project
   - Project WD2: Responsive Web Design
   - Project WD3: Interactive Web Application
   - Project WD4: Personal Portfolio Website (Current Project)
   - Project WD5: Advanced Web Application

4. **Skills (skills.html)** - Comprehensive skills display:
   - Frontend development skills
   - Backend development skills
   - Design & UX skills
   - Tools & technologies
   - Core competencies

5. **Contact (contact.html)** - Contact and engagement features:
   - Contact information cards
   - Contact form with validation
   - FAQ section
   - Social links

## How to Use

1. **Open the website:**
   - Simply open `index.html` in your web browser
   - Or use a local server for better performance

2. **Navigate the website:**
   - Click on navigation links to visit different pages
   - Use the hamburger menu on mobile devices
   - Click "View My Work" to see projects

3. **Contact:**
   - Fill out the contact form on the Contact page
   - Use the provided contact information to reach out

## Customization

### To personalize the portfolio:

1. **Update personal information:**
   - Replace "Shivesh M" with your name throughout the files
   - Update email addresses and phone numbers in contact.html
   - Modify social media links

2. **Add your own projects:**
   - Edit projects.html to replace WD1-WD5 descriptions
   - Add project images or update showcase areas
   - Include links to live projects or GitHub repos

3. **Update skills:**
   - Modify the skills section in skills.html
   - Adjust proficiency percentages
   - Add or remove technologies

4. **Customize colors:**
   - Edit CSS variables in style.css
   - Adjust the color scheme to match your brand

5. **Add profile image:**
   - Replace the SVG placeholder in index.html with your actual photo
   - Update the `<img>` tag source
